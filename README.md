# Get-Onward-Ticket 

## Structure

```
onward-ticket/
├── index.html    # Main landing page
├── style.css     # All styles (responsive)
├── script.js     # Form interactions & smooth scroll
└── README.md
```

## Features

- ✅ Responsive navbar with sticky positioning
- ✅ Hero section with illustrated SVG
- ✅ "How it works" 3-step flow
- ✅ Booking form with validation
- ✅ About / features section
- ✅ Testimonials
- ✅ FAQ with accordion
- ✅ CTA section
- ✅ Footer with newsletter signup
- ✅ Fully responsive (mobile, tablet, desktop)

## Usage

Just open `index.html` in any browser. No build step needed — pure HTML, CSS, JS.
